
# Note-Web

A meteor react webapp.

- **Website Link:**  
    - https://noteapp.cf
- **Android Client:**  
    - https://github.com/MuhtasimTanmoy/Note-Android

## Features
- Note taking CRUD app
- Real Time Update
- DDP Client
- MongoDB
- Publish Subscribe
